## About The Project
![Product Name Screen Shot](https://raw.githubusercontent.com/PacaHat/Anipaca/refs/heads/main/public/images/banner.png "Product Screenshot")


Anipaca is an open-source anime streaming website that provides a high-quality anime viewing experience. It is a PHP application built on top of the HiAnimeAPI, offering features such as no annoying video ads, episode titles, and support for various devices and resolutions.

**Why you should use this**

* Provides a platform for browsing, searching, and watching anime
* Offers high-quality streaming in 1080p, 720p, 480p, and 360p
* Supports a wide range of devices, including PCs, laptops, tablets, mobile devices, and smart TVs
* Allows users to access anime based on genres, seasons, and more
* Potential integration with Anilist in the future
